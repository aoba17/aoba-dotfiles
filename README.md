# aoba-dotfile

## install

```
bash install.sh
```

This script installs the following:

1. Xcode Command Line Tools
1. Homebrew
1. Homebrew packages
1. dotfiles

## development

### シンボリックリンクの登録

```
cd path/to/aoba-dotfiles
ln -s /absolute/path/to/dotfiles/.zshrc ~/.zshrc
```

### リンクを確認

```
ls -la ~
```
